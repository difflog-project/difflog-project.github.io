Dependencies

To use {{site.toolname}}, you need the following software:

  • Java 1.6 or later.
  • PostgreSQL 9.3.4 or later. We recommend to build and install PostgrelSQL from its source.
  • A weighted partial MaxSAT solver. {{site.toolname}} relies on a MaxSAT solver to solve the generated weighted propositional formula. Any MaxSAT solver using input and output format specified by the MaxSAT evaluation suffices. We recommend MiFuMaX.
  • LBX.
  • (Optional) MCSls.
  • (Optional) Gurobi.
